# Mapa de Problemas de Negócio (Subdomínios)

| Problema Identificado                  | Subdomínio           | Impacto                  |
|----------------------------------------|----------------------|--------------------------|
| Baixo engajamento de clientes          | Engajamento          | Redução de receitas      |
| Processos manuais e lentos             | Operações            | Baixa eficiência         |
| Silo de informações entre produtos     | Core Banking         | Dificuldade de integração|
| Dificuldade em lançar novos produtos   | Inovação             | Time-to-market elevado   |
| Falhas em compliance                   | Compliance           | Risco regulatório        |

## Análise

- **Engajamento**: Necessidade de programas como Cashback.
- **Core Banking**: Modernização para suportar novos produtos.
- **Compliance**: Automatização e integração com processos digitais.
